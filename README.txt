أكاديمية التداول الذهبي — الحزمة الكاملة (من الصفر)
---------------------------------------------------
1) ثبتي المتطلبات: pip install -r requirements.txt
2) انسخي config/.env.sample إلى config/.env واملئي القيم
3) شغّلي:
   python -m services.tv_webhook
   python -m admin_bot.main_admin_bot
   python -m trader_bot.main_trader_bot

أيقونات المشرف المغطاة:
- فتح عقد أوبشن — إغلاق عقد
- فتح صفقة سهم — إغلاق صفقة
- تحديث سعر لحظي (من Webhook)
- خروج/دخول سريع (Mock)
- قوالب الاختراق الحقيقي/الكاذب، كسر الدعم، اختراق المقاومة (Webhook JSON)
- البحث عن عقد/صفقة
- التقارير (أوبشن/أسهم)
- إدارة الواجهة (feature flags) — عبر services
- القنوات التعليمية (روابط)

أيقونات المتداول المغطاة:
- تبويب الأوبشن (عقودي النشطة)
- تبويب الأسهم
- القناة التعليمية

ملاحظة:
- التخزين JSON بسيط كبداية — يمكن ترقيته لقواعد بيانات لاحقًا.
- تنبيهات انتهاء الأوبشن مجدولة تلقائيًا عند فتح العقد.
